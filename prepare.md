## 準備事項
<strong>個人爬眠月線裝備：</strong> 
- 登山包 
- 吸濕快乾排汗衣褲（不要帶棉質的衣服外套，最少兩套） 
- 防風外套 
- 雨衣雨褲（有些防風衣也有防雨效果，如果都沒有也可選擇垃圾袋） 
- 保暖外套（抓絨衣、羽絨衣等） 
- 運動鞋、登山鞋（運動鞋要確認具有防滑功能） 
- 身份證、健保卡 
- 餅乾（依據個人需求與空間評估攜帶） 
- 手機 
- 水 
- 1/22即能下載入山許可證(帶身分文件)


<strong>重點裝備<strong><br>
無論是否有過夜，請務必準備：<br>
- 頭燈：部分隧道是沒有光源的，有頭燈才能確保行進安全。
- 水及簡單補充熱量的零食。
- 登山杖：行走高空橋梁時，有登山杖輔助可以增加支撐點，確保平衡跟重心。
- 小塑膠袋：裝自己的垃圾，或是幫忙撿垃圾淨山。
- 雨衣褲 / 衝鋒衣 / 防風雨外套等：山區易有霧及雨，有準備可以維持身體溫度避免衣服濕掉。
- 手機 APP 下載 GPX 路線軌跡圖。
- 其他裝備，就依自己安排的行程是否過夜，或依照商業團有提供甚麼

<strong>健行注意事項<strong><br>
請衡量自身體力、肌耐力、登山經驗，該行前鍛鍊就訓練，眠月線路線雖不用爬升高度，但會需要通過坍崩路段與數座高架橋，若是過夜行程則需要背負重量。<br>
雨天、颱風天等建議不要前往，一號明隧道坍崩路段，可能會形成水瀑，導致通過難度更高，另外下雨天也會造成部分路段積水以及鐵軌溼滑。<br>
過夜行程，基本水源請備足，石猴車站有 1 處活水源，其他水源點要看季節及是否有山友更新資訊分享。<br>
通過坍崩路段、行走高空橋梁時，都要單向通行，與對向人或隊伍禮讓互行，才不會發生擦撞意外。<br>
-

<strong>無痕山林 ( Leave No Trace，簡稱 LNT )<strong><br>

 👇 請務必知道、了解、遵守 "無痕山林 LNT 原則"：<br>
- 事前充分的規劃與準備 ( Plan Ahead and Prepare )
- 在可承受地點行走宿營 ( Travel and Camp on Durable Surfaces )
- 適當處理垃圾維護環境 ( Dispose of Waste Properly )
- 保持環境原有的風貌不取走任何物件 ( Leave What You Find )
- 減低用火對環境的衝擊 ( Minimize Campfire Impacts )
- 尊重保育自然環境及野生動植物 ( Respect Wildlife )
- 考量其他的使用者 ( Be Considerate of Other Visitors )

※「沼平車站」有水泥廁所，「塔山車站」「石猴車站」則是生態廁所，若忍不住需要在其他地方上廁所，請務必將垃圾帶下山。
